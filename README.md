
## Overview

This repos contains the backend for a client web application built using **Strapi CMS**. The backend provides a content management system that allows administrators to easily manage and update website content without modifying the codebase.

The API handles data storage, media management, and content delivery to the frontend application. It integrates with services such as **Cloudinary for media storage** and is deployed on **Heroku** for hosting and scalability.

This backend powers the client-facing website by exposing structured APIs that the frontend consumes to dynamically render content.

## Tech Stack

- Node.js
- Strapi CMS
- Cloudinary
- Heroku
- JavaScript
